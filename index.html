ABOUT ME

Graduate in systems engineering with strong knowledge and foundations in algorithms, data structures, object-oriented programming, relational databases, networking, web development, and management of software projects.

Through self-learning and personal projects I have acquired some extra experience in Python, Golang, PHP, Excel VBA. Also have used a bit of C and C++ while experimenting with the graphical server Xorg and its library XLib.

Like many beginners, in the process of debugging errors which come up while setting up a development environment, I obtained a basic understanding of tools like Git, Docker, AWS, and Google Cloud Platform.
